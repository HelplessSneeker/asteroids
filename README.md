## About The Project

  This Asteroids game was Programmed following the boot.dev course using the pygame labrary

### Prerequisites

  To execute the project and install the dependencies [uv](https://github.com/astral-sh/uv) Package and Project manager will be needed

### Usage

1. Clone the repo
   ```sh
   git clone https://github.com/HelplessSneeker/asteroids.git
   ```
2. then run the main file
  ```sh
   uv run main.py
```
